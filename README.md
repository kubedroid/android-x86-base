# android-x86-base

[![Docker Repository on Quay](https://quay.io/repository/kubedroid/android-x86-base/status "Docker Repository on Quay")](https://quay.io/repository/kubedroid/android-x86-base)

This repository contains scripts which create Docker images which contain filesystem of
Android-x86. The filesystem is extracted from the disk images (`.iso` files) which are
published by the Android-x86 project.

These Docker images are intended to be used by the KubeDroid build process.

## About

This repository is part of [KubeDroid](https://github.com/kubedroid). You can use KubeDroid to run Android-x86
emulators inside Kubernetes clusters, using [KubeVirt](https://kubevirt.io)

KubeDroid is sponsored by [Quamotion](http://quamotion.mobi). Quamotion provides test automation software for
mobile devices.
